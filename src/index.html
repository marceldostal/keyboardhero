Click "Start" to select an level.
After the game starts you can see objects coming from the top, the goal is to press the associated button when the object is closest to the red line. You get more points if you hit the key at the right time.

The game is playable with a touchscreen/mouse or keyboard.
Keyboardlayout can be viewed/changed at the "Options" menu.
